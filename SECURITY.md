# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities **privately** by email to
<aurig@logimentor.com>.

**Do not open a public issue for a security vulnerability.** Public issues are
for non-security bugs and feature requests only.

When reporting, include enough detail to reproduce the issue (affected version,
inputs, and observed behavior). We will acknowledge your report and keep you
informed as we investigate and address it.
